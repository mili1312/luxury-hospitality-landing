# Luxury Hospitality Landing Page

A modern, responsive landing page designed for the hospitality industry.  
Built with clean HTML, CSS, and vanilla JavaScript.

## Features

- Fully responsive layout
- iOS-style mobile navigation drawer
- Sticky header
- Hero background layering
- Contact form (front-end validation)
- Smooth scrolling
- Active section highlighting

## Tech Stack

- HTML5
- CSS3 (Custom Properties / Responsive Grid)
- Vanilla JavaScript

## Project Structure

index.html – Layout structure  
styles.css – Responsive styling and design system  
script.js – Navigation & UI interactions  

## What I Focused On

- Clean layout architecture
- Mobile-first responsiveness
- Minimal luxury design aesthetic
- UI interaction logic without frameworks
